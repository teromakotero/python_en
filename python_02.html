Challenge Python_from_Scratch_00

In the first challenge, we respect the traditions by doing "Hello, world!" program. In this challenge, we do not yet use the Player Learning method, but solve the challenge by following the instructions. The challenge is definitely easy to solve. The purpose is to familiarize yourself with the programming environment and its operation.

Instructions

  1. Type below the text: # KIRJOITA TÄMÄN RIVIN ALAPUOLELLE OMA KOODISI this Statement:

    print("Hello, world!")

  2. Run the program with ►.

Challenge Python_from_Scratch_02

In this challenge, the task is to make Pycat slip into the black cat-shaped hole. In this task, the Player Learning method is applied to find the right command and the right coordinates.

Instructions

  1. This challenge requires coordinates for help. Put the correct coordinates in the correct statement (you will find a list of Turtle commands under "Table of commands" below this page).
  2. Type below the text: # KIRJOITA TÄMÄN RIVIN ALAPUOLELLE OMA KOODISI this Statement:

    pycat.write("Hello, world!")

  3. Run the program with ►.
