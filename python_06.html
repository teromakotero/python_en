Challenge Python_from_Scratch_06

In this challenge you will learn how to make a program that tells whether a positive or negative number has been given. The program stops if a negative number is given. In this challenge, zero is considered a positive number.

We use the Player Learning method in this challenge. You can come up with a solution through trials and mistakes. The method of trial and error is one of the basic methods for solving a problem.

Instructions

  1. We need a command where user is prompted for the number with "Enter the number" In documentation you can find one.
  2. If the number is positive or zero, Pycat writes "It's a positive number".
  3. If the number is negative, Pycat writes "Negative number" and the program stops.
  4. Tip: The number from the user should be converted from a string to an integer so that the program works. In documentation you can find more information.
  5. Type below the text: # KIRJOITA TÄMÄN RIVIN ALAPUOLELLE OMA KOODISI
  6. Run the program with ►.

Challenge Python_from_Scratch_07

In this challenge you get Pycat to say "Stone!" and stop when it touches a stone figure.

We use the Player Learning method in this challenge. The feeling of frustration and irritation is an important part of the learning process.

Instructions

  1. Animate first so that Pycat walks
  2. Tip: You can use the coordinates of the characters to know when the characters touch each other.
  3. Tip: It is enough to follow the x-coordinates of pycat and stone.
  4. The location of the stone is pre-programmed so that it changes place each time you run the program.
  5. Type below the text: # KIRJOITA TÄMÄN RIVIN ALAPUOLELLE OMA KOODISI
  6. Run the program with ►.

Challenge Python_from_Scratch_08

In this challenge, the program asks "What is your name?" And then writes "Hello" and the name. If you write instead of the name "end", the program says "Goodbye" and stops.

We use the Player Learning method in this challenge. You can come up with a solution through trials and mistakes. The method of trial and error is one of the basic methods for solving a problem.

Instructions

  1. This challenge is quite similar to challenge 06, so you can take advantage of it if you do not otherwise get a solution.
  2. Type below the text: # KIRJOITA TÄMÄN RIVIN ALAPUOLELLE OMA KOODISI
  3. Run the program with ►.
