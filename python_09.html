List

A list is a tool for storing a large amount of information at once. It can be defined as a variable with other variables inside. The list consists of number-target pairs. Each item can be accessed by a number (calculation starts from zero).

Examples of Lists in Python

The list is easy to create. Give it the name and then the values inside the brackets. Below is a blank list called "list".

list = []

Use this command to add an item (in this case integer 8) to the end of the list.

list.append(8)

This command deletes and returns the last item.

list.pop()

This command places the item in a specific location in the list.

list.insert(location,item)

Use this command to remove a specific item from the list.

list.remove(item)

Use this command to select an item from a specific location in the list.

list.index(location)

Use this command to find out the length of the list.

len(list)

This command tells you how many times an item appears in the list.

list.count(item)
