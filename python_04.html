Loop Challenges

Challenge Python_from_Scratch_03

In this challenge, you will learn how to dance Pycat forever. Learn how to move Pycat back and forth. Learn to use the infinite loop.

In this challenge, the Player Learning method is used. The solution can be found through trial and error. The method of trial and error is the basic method of problem solving.

Instructions

  1. A Statement is needed to make the character (Pycat) move forward (and backward). You can find it at the bottom of the page in the Table of commands or by looking at the Statement section.
  2. You need a statement to make the program to wait for a while. The movement is so fast that it can be difficult to detect. There is a command, which can be found here Python documentation.
  3. The code should be placed in the loop so that the movement continues until the program is stopped.
  4. Type below the text: # KIRJOITA TÄMÄN RIVIN ALAPUOLELLE OMA KOODISI this Statement:

    print("Hello, world!")

  5. Run the program with ►.

Challenge Python_from_Scratch_04

This challenge teaches you how to make an animation. This is the most challenging task so far.

In this challenge, we use the Player Learning method. The feeling of frustration and irritation is an important part of the learning process in a difficult challenge.

The challenge is to animate Pycat so that it walks from left to right on the stage.

When performing this task, you understand the principle of how to make animations with Python.

Instructions

  1. Use the skills you have learned so far to solve the task.
  2. Tip:Two characters are used in this task. You can find their names by examining the code above: # KIRJOITA TÄMÄN RIVIN ALAPUOLELLE OMA KOODISI Repeatedly changing, displaying and hiding them while moving the character will make the character animated.
  3. Type below the text: # KIRJOITA TÄMÄN RIVIN ALAPUOLELLE OMA KOODISI this Statement:

    print("Hello, world!")

  4. Run the program with ►.

Challenge Python_from_Scratch_05

In this challenge you will learn how to increase the interactivity of the project. The challenge is to make a program that asks three times "What is your name?" and the Pycat writes "Hi" and the name on the scene.

Instructions

  1. You need a command to ask the user for the name and get the character to write it on the stage. In documentation you can find one.
  2. A command is required to display the text on the scene desired time.
  3. A command is needed to erase the script before a new one appears. From documentation you can find one.
  4. The code must be placed in the loop so that it is repeated 3 times.
  5. Type below the text: # KIRJOITA TÄMÄN RIVIN ALAPUOLELLE OMA KOODISI this Statement:

    print("Hello, world!")

  6. Run the program with ►.
