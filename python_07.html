Variable

A variable is a storage location for a value in the computer's memory. In Python, the variable is easy to do. Name the variable (in Python, the variable name is started with a lowercase letter, it cannot start with a number, and there can be no spaces in the name). In a word consisting of more than one word, a "_" character is inserted between the words, a "=" character and then a value to be given to the variable. There are both local (local) and general (global) variables. They come at the moment when studying how to use the function in Python.

Examples of Variables in Python

This command assigns a value of 3 to the variable number.

number = 3

This command can be used to change the value of a variable number one greater, or four.

number += 1

This command gives the variable name the value Matti.

name = "Matti"

Note that the string value requires quotes around the value.

Use this command to add another name to the variable name (note the blank space must be taken into account when entering a value)

name += " Meikäläinen"

This will show you how the space is first forgotten and then corrected.
