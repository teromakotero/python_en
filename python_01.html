Statement

In programming, Statement is a command or instruction that the programmer gives to the computer. The program is based on Statements that the programmer gives to the computer the instructions that the computer then executes. The computer program therefore consists of a number of Statements.

Examples of Statements in Python

This Statement

print("Hi!")

Print "Hi!" when the computer program is run.

print("") Statement prints the text written between quotation marks.


This Statement
object_name.forward(10)

moves the object 10 pixels forward. The amount is expressed in pixels. If you type a negative number (like -10), the object goes in the opposite direction.


object_name.back(10)

do the same thing.
