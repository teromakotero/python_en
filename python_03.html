Loop

In programming, Loops can be used to repeat the same phrases many times. Python has different possibilities to create loops.

Examples of Loops in Python

Infinite while Loop

This loop repeats the indented code below, or until the program is stopped.

while True:

After while can be a condition. Loop continues as long as the condition is true.

for Loop

This loop repeats a certain amount of times. In this case, the commands within the loop are repeated ten times.

for i in range(10):
