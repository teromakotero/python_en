Condition

In programming, conditions allow the program to work as desired when the conditions are met. The conditions are either (true) or not (false). If the condition is met, something happens. Thus, conditions can be used to break the execution of the program. Conditions can also be inserted into each other and thus make more complex branches. There are different possibilities in Python to create conditions.

Examples of Conditions in Python

If the condition is met, this if statement drives the code block below it.

			if condition is met:
				something happens

This if else statement drives the intended statements below it if the condition is met. Otherwise, it drives the intended statements below the else statement.

			if condition is met:
				something happens	
			else:
			 	something else happens

If there are more conditions, it is done in Python this way (elif is an abbreviation of else if).

			if condition is met:
    				something happens	
			elif second condition:
    				something else happens	
			elif etc. condition:
    				something else happens	
			else:
    			 	something else happens
